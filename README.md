# Final-Project-COMPSCI-132
Project Description:
	This project is my implementation of the classic higher or lower game. This game has four modes to suit whichever difficulty suits your needs from easy which should have a high rate of success, to impossible, which has a ~10% chance of success on any given attempt and has some sneaky elements in it. It filters out all non-numbers at a blistering fast O(n) time complexity playing speed.


To play: Just press run file and read the text boxes
